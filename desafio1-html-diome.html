Desafio de Projeto HTML – Estrutura Completa

O objetivo deste desafio é criar uma página HTML utilizando diversas tags aprendidas nas aulas. Você irá explorar tags para formatação, tags para listas, links e muito mais.

Algumas palavras podem ser destacadas com marcações importantes, ou com texto riscado para indicar alterações.

Também é interessante usar abreviações como HTML para explicar siglas.
